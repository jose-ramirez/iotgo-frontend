About IoTgo

IoTgo is an open source IoT platform, like WordPress, ZenCart and all other open source software, you can deploy your own IoTgo cloud platform.

ITEAD Studio is committed to provide a complete set of hardware for the cloud platform with open source hardware designs and open source firmware code.

About ITEAD

ITEAD Studio is a hardware innovation platform for makers to grow inspirations into differentiating products. Our goal is to make innovation easier.
